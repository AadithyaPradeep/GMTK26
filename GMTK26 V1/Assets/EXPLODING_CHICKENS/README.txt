This RAR file contains the following files:

STEPS: contains various footstep sounds with different pitches for some variety.

CHICKEN: contains: IDLE: various chicken sounds to be played randomly by chickens on screen; DROP_SOUND: sound to play when a chicken is dropped; GRAB: 3 sound variations to play when a chicken is grabbed.

EXPLODING: contains a "tick-tock" sound that can be looped when an explosive chicken is on screen, and EXPLOSION sounds with varying pitches that can be played when a chicken explodes.

Finally, the "OST loop" file is the background music, designed to loop. Ideally, it would be fun if the sound played faster as the game progressed (increasing the BPM by speeding up the file playback?).

All files are 44.1 kHz / 16-bit.
